# Cats Market — React + TypeScript + Ant Design

Небольшой интернет-магазин кошек 🐱: список «товаров» (котов) с пагинацией, сортировкой, карточками и авторизацией. Фронтенд общается с собственным Node.js/Express API (отдельный репозиторий).

---

## ✨ Возможности

- **Авторизация**
  - Регистрация и вход по логину/паролю.
  - JWT-токен сохраняется в cookie (`token`), фронт читает его и использует для доступа к защищённым страницам.
  - Для быстрого входа с дефолтным API можно использовать демо-аккаунт:
    - `username`: `Dmitry`
    - `password`: `11111111`
- **Приватные разделы**
  - Страница со списком котов доступна только авторизованным пользователям (`PrivateRoute` + React Router v6).
- **Список котов**
  - Пагинация (по 5 котов на страницу).
  - Сортировка по нескольким полям (имя, цена, остаток, бренд, рейтинг, количество отзывов, штрихкод).
  - Карточка кота: фото, описание, цена (с учётом скидки), порода, остаток на складе, рейтинг, число отзывов, штрихкод.
- **Состояния загрузки и ошибок**
  - Спиннер на время загрузки данных.
  - Обработка сетевых/серверных ошибок при запросах.
- **UI/UX**
  - Ant Design компоненты + `styled-components` для кастомной стилизации.
  - Поддержка сортировки и пагинации через URL-параметры (`page`, `sort`).

---

## 🧰 Стек

- React 18, TypeScript
- Vite (dev-сервер и сборка)
- React Router v6
- Ant Design, `styled-components`
- Кастомные хуки (`useProducts`) для загрузки и сортировки данных
- Axios для работы с API
- **Архитектура: Feature-Sliced Design (FSD)**

---

## 🧱 Архитектура

Проект организован по упрощённой схеме FSD: слои `app`, `pages`, `entities`, `shared` без избыточных уровней для маленького проекта.
